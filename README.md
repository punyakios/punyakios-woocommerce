# PunyaKios Payment Gateway for WooCommerce

Accept automated QRIS payments through the PunyaKios merchant service directly on your WooCommerce store.

## Features
- **Automated QRIS Generation**: No manual checking required.
- **Real-time Notifications**: Order status updates automatically upon payment.
- **Secure Redirect**: Customers pay on a mobile-friendly secure page.
- **Easy Setup**: Just enter your Merchant API Key.

## Installation
1. Download the plugin folder.
2. Upload to your WordPress plugins directory (`/wp-content/plugins/`).
3. Activate the plugin via the 'Plugins' menu.
4. Go to **WooCommerce > Settings > Payments**.
5. Enable **PunyaKios (QRIS)** and click **Manage**.
6. Enter your **Merchant API Key** from the PunyaKios dashboard.

## Requirements
- WordPress 5.0+
- WooCommerce 3.0+
- PHP 7.4+
